# maestro-luis-apis
luis alejandro avila lopez
Módulo de Geolocalización
Integra un mapa interactivo que despliega la localización exacta del campus universitario, sirviendo como guía visual para ubicar las instalaciones y las vialidades colindantes.

Módulo de Feed Social
Despliega un flujo continuo de mensajes y contenido compartido por la comunidad estudiantil, recreando la dinámica y la interacción de un entorno de red social.

Módulo de Consulta Estructurada
Conecta directamente con el servidor para extraer información almacenada, presentando los registros, campos y tablas mediante un formato visual limpio y ordenado.
